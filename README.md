# Ethiopian ↔ Gregorian Calendar Converter

A simple Python program that converts dates between the **Ethiopian Calendar (EC)** and the **Gregorian Calendar (GC)**.  
It supports both directions:

- GC → EC  
- EC → GC  

Leap years are handled for the Gregorian calendar.

---

## How to Run
1. Install Python  
2. Save this file as `calendar_converter.py`  
3. Run:
   ```
   python calendar_converter.py
   ```
